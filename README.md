# UltimateBot
My First best AI minecraft bot on node js
## ✨ Features

### 🧠 Smart Decision Making
- **GOAP (Goal-Oriented Action Planning)** — 35+ dynamic goals with priorities that adapt to the situation
- **Full progression** — Wood → Stone → Iron → Diamond → Nether → Ender Dragon
- **Autonomous survival** — Eats when hungry, fights when threatened, sleeps when safe

### 😃 Human-Like Behavior
- **Continuous head sway** — never stands perfectly still
- **Two-step looking** — turns naturally, not instantly snapping
- **Random fidgeting** — switches hotbar slots, glances around
- **Idle animations** — crouch spam, random jumps, looking at mobs

### 🏠 Intelligent Building
- **Random house generation** — every home is unique (5-8 width, 3-4 height, random windows/doors)
- **Gradual improvements** — fences, gardens, torches, animal pens
- **Auto-repair** — checks and fixes damage every time you return home
- **Storage management** — deposits valuables into chests

### ⛏️ Safe & Efficient Mining
- **Raycast line-of-sight** — never digs through walls
- **Branch mining** — automatic tunnel digging at Y=11
- **Danger avoidance** — detects lava, cliffs, and drowning
- **Tool management** — equips best pickaxe, crafts when needed

### ⚔️ Advanced Combat
- **W-TAP & S-TAP** — real PvP techniques for knockback control
- **Critical hits** — jump-attacks like real players
- **Strafe and dodge** — moves unpredictably in combat
- **PVP self-defense** — only fights back when attacked
- **Bow shooting** — aims with prediction for moving targets

### 🔥 Endgame Ready
- **Nether portal** — builds and lights portal automatically
- **Ender preparation** — hunts endermen, crafts eyes of ender
- **Stronghold navigation** — throws eyes, finds portal room
- **Dragon fight** — destroys crystals, uses bow and melee

### 🖥️ Easy to Use
- **Windows installer (.exe)** — no Node.js, no Python, no coding
- **Web dashboard** — monitor bot in real-time at `http://localhost:3000`
- **Chat commands** — control the bot directly from Minecraft chat
